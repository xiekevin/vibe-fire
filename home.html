about
VibeFire is a project created for the purpose of exploring basic website design using CSS, HTML, and jQuery as a final submission for the web design decal. The idea behind VibeFire is for it to emulate a music recommendation site (like Pandora or Spotify) by allowing users to type in a song, genre, or artist and recieve music recommendations based on what is entered.

As the first website I've designed, there are many design inconsistencies and styling glitches, but nevertheless it's been a great first step into furthering my passion and expertise in design. While creating the website, significant difficulty was encountered in trying to getting elements to adjust with screen resolution, positioning elements using margin or padding, centering elements horizontally and/or vertically, and getting the proper alignment.
